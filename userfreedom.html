3. User Control and Freedom

Users often choose system functions by mistake and will need a clearly marked "emergency exit" to leave the unwanted state without having to go through an extended dialogue. Support undo and redo.

Usually, in order to facilitate the navigation of users website designers create a home page button on each page. This button is called "emergency exit", and it is a simple way to let users feel in control of your site.
